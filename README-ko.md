# Citrine

[English](READM.md) ・ [日本語](README-ja.md) ・ 한국어

Citrine은 귀엽고 똑똑한 BotBone 챗봇 플러그인입니다.

## 사용법

우선 빌드합니다.

```
dotnet build
```

BotBone의 /plugins 폴더에 [Citrine.dll](bin/Debug/netstandard2.1/publish/Citrine.dll)를 복사하고 BotBone을 실행
